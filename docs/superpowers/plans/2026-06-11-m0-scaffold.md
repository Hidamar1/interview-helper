# M0 脚手架与主题成型 实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 搭建可运行的 Next.js 16 项目空壳：设计 token 成型（暖橙活力风）、全局布局骨架、数据库连通、测试与 CI 基建就绪。

**架构：** 单仓库 Next.js 16 App Router 全栈应用（见 `docs/02-技术方案.md` §1）。本里程碑只搭骨架不做业务：Tailwind v4 CSS-first 设计 token + shadcn 暖橙改造 + Prisma 7 连通 Neon + Vitest/Playwright/CI 三套质量基建。

**技术栈（2026-06-11 实测版本，已复核）：**

| 包 | 版本 | 说明 |
|---|---|---|
| next | 16.2.9 | Turbopack 默认；**Cache Components 不启用** |
| react / react-dom | 19.2.7 | - |
| tailwindcss | 4.3.0 | CSS-first `@theme`，无 tailwind.config.js |
| shadcn (CLI) | 4.11.0 | ⚠️ 文档基线写 3.x，现已 4.x，init 流程以执行时 Context7/官方文档为准 |
| prisma / @prisma/client / @prisma/adapter-neon | 7.8.0 | 无 Rust 引擎；用 prisma.config.ts（v7 不再自动加载 .env） |
| zod | 4.4.3 | - |
| vitest | 4.1.8 | + @testing-library/react 16.3.2 |
| @playwright/test | 1.60.0 | - |
| pnpm | 11.5.3 | - |
| ai（M3 用，现在锁定主版本） | 6.0.200 | **决策：锁定 AI SDK v6**（技术方案预案：「若 6 已稳定则评估后锁定」，v6 已 GA 且 patch 200+，配套 @ai-sdk/react@3.0.202、@ai-sdk/deepseek@2.0.36） |

**Next.js 16 关键变更（写代码时必须遵守）：** 页面组件的 `params` / `searchParams` 是 **Promise**，必须 `await`。

"use client";

import Link from "next/link";

export function NavLink({ href, children }: { href: string; children: React.ReactNode }) {
  const pathname = usePathname();

      className={cn(
        "text-sm text-neutral-600 transition-colors hover:text-primary",

    >

  );
}

```ts
import { PrismaClient } from "@/generated/prisma/client";
import { PrismaNeon } from "@prisma/adapter-neon";

const createClient = () =>
  new PrismaClient({
    adapter: new PrismaNeon({ connectionString: process.env.DATABASE_URL! }),
  });

const globalForPrisma = globalThis as unknown as { prisma?: ReturnType<typeof createClient> };

export const prisma = globalForPrisma.prisma ?? createClient();

if (process.env.NODE_ENV !== "production") globalForPrisma.prisma = prisma;
```
